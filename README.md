# SunCart - Summer Essentials Store ☀️🛒

A modern, visually stunning eCommerce platform focused on summer essentials. SunCart allows users to browse a curated list of summer products such as sunscreen, sunglasses, and beach gear, check detailed information, and experience a smooth, fast, and responsive user interface.

## 🎯 Key Features
- **Modern UI/UX**: Aesthetic design with glassmorphism and vibrant summer colors (`#fbbc04` yellow accents).
- **Responsive Layout**: Completely mobile-ready design fitting tablets and desktops perfectly.
- **Product Details & Browsing**: Static data-driven catalog with detailed product pages and galleries.
- **Authentication System Ready**: Frontend integrated forms designed to be powered by BetterAuth (features Email/Password & Google Login options).
- **Safe Environment Variables**: Using `.env.local` for secure Firebase and auth configuration.

## 📦 Tech Stack & Packages
- **Next.js 15 (App Router)** - Core framework
- **Tailwind CSS v4** - Styling
- **Shadcn UI** - Reusable component foundation
- **lucide-react & react-icons** - Iconography
- **animate.css** (tw-animate-css) - Animation utilities
- **better-auth** - Flexible authentication (integrated functionality for signup/login)

## 🛠️ Usage
1. Clone the repository
2. Run `npm install` (with `--legacy-peer-deps` if upgrading from older Next configurations)
3. Set up environment variables in `.env.local`
4. Run `npm run dev` to start the development server
